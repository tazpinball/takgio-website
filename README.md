# TAKGIO Website

Static website for [takgio.com](https://takgio.com).

## How to update

1. Edit the HTML/CSS/JS files
2. Commit and push to `main` branch
3. Vercel auto-deploys within seconds

## Folder structure

```
├── index.html        # Main page
├── css/
│   └── styles.css    # Stylesheet
├── images/           # Images and graphics
├── js/               # JavaScript files
└── README.md
```

## Adding new pages

Create a new `.html` file in the root (e.g., `about.html`) and link to it from `index.html`.
